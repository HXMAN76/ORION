# Gantari Font Files

Place the Gantari font files in this directory:

- `Gantari-Regular.woff2`
- `Gantari-Medium.woff2`
- `Gantari-SemiBold.woff2`
- `Gantari-Bold.woff2`

You can download Gantari from Google Fonts and convert to WOFF2 format using tools like:
- https://google-webfonts-helper.herokuapp.com/fonts/gantari
- https://cloudconvert.com/ttf-to-woff2

The app will fall back to Inter or system fonts if these files are not present.
